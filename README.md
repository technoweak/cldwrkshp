# cldwrkshp

A lightweight, hands-on Cloud Workshop repository with labs, examples, and resources to learn cloud fundamentals and common workflows.

## Contents
- Overview
- Prerequisites
- Setup
- Quick start
- Directory structure
- Contributing
- License

## Overview
This repo contains exercises and example code used for cloud workshops and training. Each lab is self-contained in its own folder and includes instructions, scripts, and resources needed to complete the exercises.

## Prerequisites
- Git
- A cloud account (AWS/Azure/GCP) depending on the lab
- Recommended: Docker, Terraform, kubectl (per individual lab requirements)

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/technoweak/cldwrkshp.git
   cd cldwrkshp
   ```
2. Open the lab folder you want to follow and read its README or instructions.

## Quick start
- Each lab directory contains a README with step-by-step instructions.
- Follow the lab-specific prerequisites and setup steps before running scripts.

## Directory structure (example)
- labs/         - individual workshop labs
- scripts/      - utility scripts used by labs
- examples/     - sample code and templates
- docs/         - workshop guides and reference material
